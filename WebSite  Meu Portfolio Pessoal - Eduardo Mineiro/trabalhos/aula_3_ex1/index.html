O QUE SÃO TAGS SEMANTICAS

Objetivo

 A maior mudança proposta na linguagem HTML5 foi na sua semântica.

Semântica diz respeito ao significado do elemento. Os elementos semânticos descrevem seu significado para o navegador e para o desenvolvedor.O principal objetivo das tags semânticas é a organização semântica do documento. Mas o comportamento padrão (que poderá ser alterado com estilos CSS) será de blocos apresentados em posições verticais diferentes.


Entenda um pouco mais:

 Elementos de Semântica – de Bloco São Blocos de Conteúdo, como qualquer DIV, mas eles servem para dizer qual é o sentido de cada Bloco. Uma estrutura possível de utilização desses blocos, que nos ajuda a entender o que eles representam.

DEFINIÇÕES
  • 1) Conteúdo independente do resto do site

  • 2) Mediação de conflitos familiares

  • 3) Cabeçalho de uma seção ou artigo

  • 4) Agrupamento de cabeçalhos ( h1 , etc)

  • 5) Rodapé de uma seção ou artigo

  • 6) Região com os links para navegação do site

  • 7) Geralmente usado com algum conteúdo relacionado

  • 8) Agrupa imagens com sua legenda
